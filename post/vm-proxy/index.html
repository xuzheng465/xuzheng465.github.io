Parallels Desktop中的虚拟机使用宿主机代理

2022-01-26

Contents

    • 环境
    • 配置
    • Reference

环境

宿主机：macOS Monterey 12.1

虚拟机：CentOS 8

配置

  1. Clash勾选上Allow connect from LAN

  2. 点击Copy shell command 看看端口号

    1

    export https_proxy=http://127.0.0.1:7890 http_proxy=http://127.0.0.1:7890 all_proxy=socks5://127.0.0.1:7890

  3. 在宿主机的终端输入ifconfig查看宿主机的IP地址。 就是en0下面inet 后面的IP地址。比如是 192.168.37.1

  4. 在虚拟机Settings>Network>Network Proxy中选择Manual

  1. 在虚拟机终端编辑.bashrc文件，加入

function proxy_on() {
    export no_proxy="localhost,127.0.0.1,localaddress,.localdomain.com"
    export http_proxy=http://xx.xx.xx.xx:7890
    export https_proxy=http://xx.xx.xx.xx:7890
    export all_proxy=socks5://xx.xx.xx.xx:7890
    echo -e "已开启代理"
}
function proxy_off() {
    unset http_proxy
    unset https_proxy
    unset all_proxy
    echo -e "已关闭代理"
}

xx.xx.xx.xx处替换成宿主机的IP的地址

  1. 虚拟机网络模式选择Bridged Network (WiFi)
